# butter-tteok-map

버터떡 맵은 **요즘 뜨는 버터떡 판매처를 빠르게 찾고 공유하기 위한 Apps in Toss 미니앱**입니다.

트렌드성 먹거리의 특성상 핵심은 완벽한 기능보다 **빠른 발견, 가벼운 탐색, 쉬운 공유**입니다.

## What it solves

- 버터떡이 어디서 파는지 찾기 번거롭다
- SNS/커뮤니티에서 본 가게 정보를 다시 찾기 어렵다
- 유행이 빠르게 변해서 정보가 금방 흩어진다

## Core Features

- 📍 버터떡 판매처 탐색
- 🗺️ 지도 기반 위치 확인
- 🔎 빠른 검색 및 탐색
- 📱 Toss 미니앱 환경에 맞춘 가벼운 사용 경험

## Tech Stack

- TypeScript
- Vite
- Tailwind CSS
- Apps in Toss mini app runtime

## Status

**MVP / Active**

현재 목표는 빠르게 배포하고 실제 반응을 확인하는 것입니다.

## Project Goal

이 프로젝트는 단순한 지도 앱이 아니라,
**짧게 뜨는 바이럴 키워드를 빠르게 제품으로 바꿔 반응을 검증하는 실험**입니다.

즉,
- 트렌드 포착
- 빠른 MVP 제작
- 실제 사용자 반응 확인
- 확장 가능성 검증

이 흐름을 가장 짧게 돌리는 데 초점을 맞추고 있습니다.
